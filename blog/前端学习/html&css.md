# html&css 笔记

## 标签

1、文本标题（h1-h6）

```html
<h1>一级标题</h1>
```

> 注:文本标题标签自带加粗，有自己的文本大小，并且独占一行，有默认间距

2、段落文本(p)

```html
<p>段落文本内容</p>
```

标识一个段落(段落与段落之间有段间距)

3、换行(br)

```html
<br / >
```

换行是一个空标记(强制换行)

4、水平线

```html
<hr />空标记
```

5、加粗有两个标记(推荐strong)

```html
<b>加粗内容</b>只是显示加粗
<strong>强调的内容</strong>突出的文本
```

6、倾斜有两个标记(推荐em)

```html
<em>强调文本</em>
<i>倾斜文本</i>
```

7、删除线有两个标记(推荐del)

```html
<s>文本</s>删除线
<del>文本</del>删除线
```

8、扩展

```html
<u>文本</u>下划线
<sub> </sub>下标
```
